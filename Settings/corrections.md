# Corrections

This file helps the transcription engine get names, terms, and phrases right.
Edit it any time you notice a recurring mistake in your transcripts.

## Terms

Words and names the transcriber should recognize correctly.
Add one per line with a dash prefix.

- Phenom

## Replacements

When the transcriber consistently gets a word wrong, add the mistake and
the correct version here. Matching is case-insensitive.

| Heard | Should be |
|-------|-----------|
| fennom | Phenom |
| phenam | Phenom |
